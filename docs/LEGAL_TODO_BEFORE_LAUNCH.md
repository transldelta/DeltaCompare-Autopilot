# Rechtliche TODOs vor dem Launch

> ⚠️ Diese Liste ersetzt keine Rechtsberatung. Lass deine Texte vor Veröffentlichung
> von einem Anwalt oder einer auf IT-Recht spezialisierten Kanzlei prüfen.

## Impressum

- [ ] Vollständiger Name oder Firmenname eingetragen
- [ ] Ladungsfähige Anschrift (kein Postfach)
- [ ] Telefon und/oder Schnellkontakt-E-Mail
- [ ] Umsatzsteuer-ID, falls vorhanden
- [ ] Bei journalistischen Inhalten: Verantwortlich nach § 18 Abs. 2 MStV
- [ ] OS-Plattform-Link der EU-Kommission ist enthalten
- [ ] Klare Aussage zur Streitschlichtung (Teilnahme/Nicht-Teilnahme)

## Datenschutzerklärung

- [ ] Verantwortlicher mit Kontaktdaten benannt
- [ ] Zugriffsdaten / Server-Logs beschrieben (Art. 6 Abs. 1 lit. f DSGVO)
- [ ] Cookies erklärt (notwendige vs. einwilligungsbedürftige)
- [ ] Genutzte Tracking-Dienste konkret aufgeführt (GA4, GTM, Meta Pixel, TikTok Pixel)
- [ ] Affiliate-Netzwerke nennen (Awin, financeAds, CHECK24 etc.)
- [ ] Hosting-Anbieter und Server-Standort genannt
- [ ] Newsletter-Anbieter genannt, falls vorhanden
- [ ] Speicherdauer der Daten dokumentiert
- [ ] Betroffenenrechte (Auskunft, Berichtigung, Löschung etc.) erläutert
- [ ] Zuständige Aufsichtsbehörde genannt

## Affiliate-Hinweis

- [ ] Sichtbar im Footer auf jeder Seite
- [ ] Auf Vergleichsseiten unter dem Intro sichtbar
- [ ] Auf Anbieter-Detailseiten sichtbar
- [ ] Affiliate-Buttons tragen das Label „Werbung / Affiliate-Link“
- [ ] Buttons setzen `rel="sponsored nofollow"` (bereits eingebaut)

## Werbe-Kennzeichnung

- [ ] Auf Social Media werden Affiliate-Posts mit „#Werbung" oder „#Anzeige" gekennzeichnet
- [ ] Bezahlte Anzeigen tragen das Plattform-Label „Gesponsert"
- [ ] In Newslettern wird ein Hinweis zu Affiliate-Links eingebaut

## Beratungsausschluss

- [ ] Auf Vergleichsseiten und Anbieter-Seiten ist klar formuliert,
      dass die Inhalte **keine** Finanz-, Steuer-, Versicherungs- oder Rechtsberatung sind
- [ ] Im Haftungsausschluss steht derselbe Hinweis
- [ ] Bei sensiblen Themen (PKV, Kredit, Steuer) zusätzlich „Bitte individuell prüfen lassen“

## Cookie- / Tracking-Hinweise

- [ ] Consent-Banner sichtbar beim ersten Seitenaufruf
- [ ] „Ablehnen“ und „Akzeptieren“ gleichwertig prominent
- [ ] Tracking-Skripte laden **erst nach Einwilligung** (bereits implementiert)
- [ ] Einwilligung jederzeit widerrufbar (z. B. über separaten Link im Footer)
- [ ] Empfehlung: zusätzliches Consent-Tool wie Usercentrics oder CookieFirst
      einsetzen, sobald echte Tracker eingebunden sind

## Newsletter (falls aktiviert)

- [ ] Double-Opt-In eingerichtet
- [ ] Abmelde-Link in jedem Newsletter
- [ ] Impressum-Link in jedem Newsletter
- [ ] Datenschutz-Hinweis im Anmelde-Formular
- [ ] Liste der genutzten Newsletter-Tools in der Datenschutzerklärung

## Markenrechte

- [ ] Keine fremden Logos ohne Lizenz/Erlaubnis
- [ ] Keine fremden Marken-Slogans in Texten
- [ ] Keine fremden Markenwörter in Google-Ads-Kampagnen, sofern nicht ausdrücklich freigegeben

## Verträge mit Partnern

- [ ] Affiliate-Programm-Verträge gelesen und akzeptiert
- [ ] Eigene Rolle (Steuerlich: gewerblich? freiberuflich?) mit Steuerberater geklärt
- [ ] Umsatzsteuer-Behandlung der Provisionen geklärt
- [ ] Aufbewahrung der Abrechnungen (10 Jahre) eingerichtet

## Bevor du auf „Veröffentlichen“ drückst

- [ ] Anwaltliche Prüfung Impressum + Datenschutz + Affiliate-Hinweis
- [ ] Steuerliche Anmeldung als Affiliate-Publisher (sofern gewerblich)
- [ ] Buchhaltung für Affiliate-Einnahmen eingerichtet
- [ ] DSGVO-Verzeichnis der Verarbeitungstätigkeiten erstellt

**Erst danach Sitemap einreichen und Marketing-Aktivitäten starten.**
