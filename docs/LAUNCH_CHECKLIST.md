# Launch-Checkliste

Bevor du DeltaCompare live nimmst, gehe die folgenden Punkte durch.

## Inhalt

- [ ] Demo-Anbieter durch echte Anbieter ersetzen oder klar als Beispiele kennzeichnen.
- [ ] Alle Affiliate-Links (statt `DEMO_LINK_ERSETZEN`) einsetzen.
- [ ] Mindestens 10 Anbieter mit echten Bewertungen pflegen.
- [ ] Vergleichsseiten auf Tippfehler prüfen.
- [ ] Mindestens 5 Ratgeberartikel mit eigenem Mehrwert.

## Rechtliches

- [ ] Impressum vollständig (Anwalt prüfen lassen).
- [ ] Datenschutzerklärung individuell angepasst.
- [ ] Affiliate-Hinweis auf allen Vergleichsseiten und im Footer.
- [ ] Haftungsausschluss vorhanden.
- [ ] Cookie-/Consent-Banner integriert.
- [ ] Newsletter DOI eingerichtet (falls Newsletter geplant).

## Tracking

- [ ] `NEXT_PUBLIC_SITE_URL` auf finale Domain umgestellt.
- [ ] Google Analytics ID gesetzt.
- [ ] Google Tag Manager (optional).
- [ ] Meta Pixel (falls Meta Ads geplant).
- [ ] TikTok Pixel (falls TikTok Ads geplant).
- [ ] `AUTH_SECRET` als starker Zufallsstring in der Produktion.
- [ ] `ADMIN_PASSWORD` ersetzt.

## Technik

- [ ] `npm run build` läuft fehlerfrei.
- [ ] Datenbank in der Produktion eingerichtet (PostgreSQL empfohlen).
- [ ] Backups eingerichtet.
- [ ] SSL aktiv (HTTPS).
- [ ] Domain auf Deployment zeigt.

## SEO

- [ ] Sitemap unter Google Search Console eingereicht.
- [ ] OG-Bild in `public/og/default.svg` an dein Branding angepasst.
- [ ] Title und Meta-Description aller Top-Seiten gepflegt.

## Affiliate-Konten

- [ ] Awin / financeAds / CHECK24 / Tarifcheck etc. registriert.
- [ ] Steuerdaten und IBAN bei jedem Netzwerk hinterlegt.
- [ ] Tracking-Subids im Admin gepflegt (optional, hilft bei Reports).

## Mobile / UX

- [ ] Mobile-Ansicht aller Seiten geprüft.
- [ ] Affiliate-Buttons groß genug und gut sichtbar.
- [ ] Ladezeiten unter 2 Sekunden.

## Marketing

- [ ] Social-Media-Konten erstellt.
- [ ] Pinterest-Profil & 30 Pins terminiert.
- [ ] Newsletter-Tool gewählt und Anmelde-Formular eingebaut.

Wenn alle Punkte abgehakt sind, kannst du Online gehen. 🎉
