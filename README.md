# DeltaCompare Autopilot

Vollständiges Affiliate- und Vergleichsportal für Selbstständige – Next.js 14, TypeScript, Tailwind, Prisma.

## Schnellstart

```bash
npm install
cp .env.example .env
npm run db:push
npm run db:seed
npm run dev
```

- Frontend: http://localhost:3000
- Admin-Login: http://localhost:3000/admin/login (Zugangsdaten in `.env`)

## Produktiv bauen

```bash
npm run build
npm start
```
